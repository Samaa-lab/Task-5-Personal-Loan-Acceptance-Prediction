# Task-5-Personal-Loan-Acceptance-Prediction
Data Science Internship Task-5-Personal-Loan-Acceptance-Prediction

## Objective
Predict which customers are likely to accept a personal loan offer and identify high-conversion customer groups.

## Dataset
Bank Marketing Dataset (UCI Machine Learning Repository)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow
1. Data Exploration
2. Age Analysis
3. Job Analysis
4. Marital Status Analysis
5. Data Preprocessing
6. Logistic Regression Modeling
7. Decision Tree Modeling
8. Model Comparison
9. Business Insight Extraction

## Models Used
- Logistic Regression
- Decision Tree Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Key Findings
- Customer demographics influence offer acceptance.
- Certain age groups and occupations have higher acceptance rates.
- Logistic Regression achieved the best balance between Recall and F1 Score.
- The model can help banks improve campaign targeting and reduce marketing costs.
